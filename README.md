## What Is This?

A mobile-first, dark-themed HTML price guide designed to be used **on your phone while digging through card bins** at Japanese flea markets (フリマ), Hard-Off, Book-Off, recycle shops (リサイクルショップ), and countryside second-hand stores.

### Features

- 📱 **Mobile-responsive** — designed for one-handed phone use while flipping through cards
- 🌙 **Dark theme** — easy on the eyes in dimly-lit shops
- 🇯🇵 **Japanese card names** (purple text) — match what's printed on the actual cards
- 💰 **Actionable buy prices** — every card has a "Buy under ¥X" tag so you know your max offer instantly
- 🔥 **Flip multipliers** — see the potential return (2x, 5x, 10x+) at a glance
- 📦 **Sealed products & loose packs** — know MSRP (定価) vs market price for boxes and booster packs
- 💡 **Quick ID tips** — 5-second checks, Japanese set symbols, price check apps
- 📶 **Works offline** — all images stored locally, no internet needed

## Usage

### Online
Visit the [live site](https://arthurb2l.github.io/japan-pokemon-card-price-guide/).

### Offline (Recommended)
1. Clone or download this repo
2. Open `index.html` in your phone's browser
3. All images are in `/images/` — no internet required

```bash
git clone https://github.com/arthurb2l/japan-pokemon-card-price-guide.git
```

## Price Sources

Prices are based on March 2026 market data from TCGPlayer, Mercari JP, and Yahoo Auctions JP. Updated periodically.
