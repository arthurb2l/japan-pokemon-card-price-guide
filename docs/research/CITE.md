# How to Cite

## BibTeX

```bibtex
@misc{brejon2026aiworktax,
  author = {Brejon, Arthur},
  title = {AI Work Tax: A Framework for Sustainable AI-Augmented Open Source Contribution},
  year = {2026},
  publisher = {GitHub},
  url = {https://github.com/arthurb2l/japan-tcg-price-guide},
  note = {Accessed: YYYY-MM-DD}
}
```

Brejon, A. (2026). *AI Work Tax: A Framework for Sustainable AI-Augmented Open Source Contribution*. GitHub. https://github.com/arthurb2l/japan-tcg-price-guide
